# Dir Maker

A simple tool to convert indented text into a beautiful directory tree structure.

<img width="3024" height="1890" alt="CleanShot 2026-01-01 at 18 44 31@2x" src="https://github.com/user-attachments/assets/79f47851-30ca-42da-b9e9-7402a54b6c4d" />

## Features

- Convert indented text to tree structure with `├──` and `└──` connectors
- Add trailing slash option for directories
- Copy formatted output with one click
- Dark/Light theme support
- Local storage persistence

## Development

### Setup

```bash
pnpm install
```

### Run Development Server

```bash
pnpm dev
```

Open http://localhost:5173 with your browser.

### Build

```bash
pnpm build
```

### Test

```bash
pnpm test
```
